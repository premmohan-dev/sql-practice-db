# SQL Practice Database

A beginner-friendly SQL practice project featuring sample database schemas, data, and queries.

## Project Structure

- **schema.sql** - Database schema with table definitions
- **sample-data.sql** - Sample data to populate the database
- **practice-queries.sql** - Example SQL queries for practice

## Database Schema

The database includes:
- **users** - User information
- **products** - Product catalog
- **orders** - Customer orders
- **order_items** - Items within each order

## Learning Objectives

- Practice basic SQL CRUD operations (Create, Read, Update, Delete)
- Learn about table relationships and foreign keys
- Practice JOIN operations
- Work with aggregate functions (COUNT, SUM)
- Group and filter data effectively

Created as a learning project for GitHub and SQL practice.
